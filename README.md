# 隐私政策

**生效日期：2025年11月20日**

感谢您使用 DCPilot 应用程序（以下简称"本应用"或"我们"）。我们非常重视您的隐私和个人信息保护。本隐私政策将帮助您了解我们如何收集、使用、存储和保护您的个人信息。

## 重要声明

本应用基于开源项目 [Docker Copilot](https://github.com/onlyLTY/dockerCopilot) 的 RESTful API 构建，旨在为用户提供便捷的 Docker 容器管理工具。

**本应用采用完全本地化的架构设计**，具有以下核心特性：

- ✓ **本地部署**：您所连接的 Docker Copilot 服务器由您在本地自主部署和管理
- ✓ **数据本地化**：所有容器数据、镜像信息、配置数据均存储在您的本地设备和服务器上
- ✓ **零第三方上传**：本应用不会主动将您的任何数据上传至第三方服务器
- ✓ **无数据分析**：本应用不会进行任何基于第三方服务的数据收集或分析
- ✓ **点对点通信**：应用仅与您配置的本地服务器进行直接通信

我们强烈建议您：
1. 妥善保管服务器地址和安全密钥
2. 确保您的本地网络环境安全
3. 定期更新应用版本以获得最新的安全特性
4. 合理使用本应用，遵守相关法律法规

## 1. 我们收集的信息

### 1.1 您主动提供的信息
在使用本应用时，您需要提供以下信息以连接您的本地服务器：
- **服务器地址**：您自主部署的 Docker Copilot 服务器地址（本地或局域网地址）
- **安全密钥**：用于身份验证和安全连接的密钥
- **配置信息**：包括容器配置、镜像设置等操作数据

**重要提示**：以上所有信息仅存储在您的设备本地，不会传输至任何第三方服务器。

### 1.2 本地收集的技术信息
为保证应用正常运行，我们可能在您的设备本地记录：
- **应用日志**：本地运行日志，用于故障排查（仅存储在您的设备上）
- **设备基本信息**：操作系统类型和版本，用于适配不同平台
- **偏好设置**：主题、语言等个性化设置

## 2. 信息的使用

本应用收集的信息仅用于以下本地功能：
- 与您自主部署的 Docker Copilot 服务器建立安全连接
- 管理和操作您本地的 Docker 容器和镜像
- 在您的设备上保存应用配置和偏好设置
- 本地诊断和优化应用性能

**我们明确承诺**：
- 不会将您的任何信息用于广告推送
- 不会将您的任何信息出售给第三方
- 不会使用第三方数据分析工具追踪您的行为
- 不会将您的服务器信息上传至任何云端服务

## 3. 信息的存储与传输

### 3.1 本地存储
- 您的服务器地址、安全密钥等敏感配置信息采用加密方式存储在您的设备本地存储空间中
- 应用数据仅保存在您的设备上，您可以随时通过清除应用数据完全删除
- 所有 Docker 相关数据存储在您自己的服务器上，由您完全掌控

### 3.2 数据传输
- 本应用仅与您配置的本地服务器进行点对点通信
- 通信过程使用加密传输协议（HTTPS/TLS）保护数据安全
- 不存在任何数据向第三方服务器的传输行为
- 您的服务器地址和密钥永远不会离开您的设备

### 3.3 数据安全
- 采用行业标准的加密技术保护本地存储的敏感数据
- 定期更新安全措施以应对新的安全威胁
- 所有安全措施均在您的设备本地执行

## 4. 信息的共享与披露

基于本应用的本地化架构特性，我们郑重声明：

**我们不会以任何形式共享您的个人信息**，包括但不限于：
- 不会出售、出租或交易您的任何个人信息
- 不会将数据提供给任何第三方广告商或数据分析公司
- 不会与任何第三方服务共享您的服务器配置信息
- 不会在未经您明确授权的情况下访问或传输您的数据

**唯一的例外情况**：
- 根据适用的法律法规、法律程序、诉讼或政府主管部门的强制性要求
- 在紧急情况下，为保护用户或公众的人身安全

## 5. 您的权利

由于本应用采用完全本地化存储，您对所有数据拥有完全的控制权：

- **完全访问权**：所有数据存储在您的设备和服务器上，您可以随时查看
- **修改权**：您可以随时在应用中修改任何配置信息
- **删除权**：您可以通过以下方式删除数据：
  - 在应用中清除配置信息
  - 卸载应用以删除所有本地数据
  - 在您的服务器上管理和删除服务器端数据
- **数据导出权**：您可以导出备份您的容器配置
- **服务器控制权**：您完全控制自己的 Docker Copilot 服务器，包括数据的保留和删除

## 6. 儿童隐私保护

本应用为专业的容器管理工具，不面向 14 岁以下的儿童。我们不会故意收集儿童的个人信息。由于本应用不涉及云端账户系统，也不会在服务端存储任何用户身份信息，因此不存在儿童信息泄露风险。

如果您是家长或监护人，发现您的孩子在未经许可的情况下使用了本应用，您可以直接卸载应用或清除应用数据。

- ✗ 无第三方社交媒体集成
- ✗ 无第三方云存储服务

我们可能会不时更新本隐私政策，以反映我们在隐私保护方面的改进或法律要求的变化。更新后的政策将在应用中发布，并注明新的生效日期。

## 9. 数据保留

基于本地化架构，数据保留由您完全掌控：

**设备端数据**：
- 您的配置信息保留在您的设备上，直到您主动删除应用或清除数据
- 本地日志文件由系统自动管理，不会无限增长
- 您可以随时通过设备设置清除应用的所有本地数据

**服务器端数据**：
- 所有 Docker 相关数据存储在您自己的服务器上
- 数据保留时间和管理方式完全由您决定
- 您可以随时在服务器上删除或备份数据

## 10. 数据传输

**本应用不涉及跨境数据传输**。所有数据通信均在您的本地网络环境中进行：
- 移动设备 ↔ 您的本地服务器
- 不涉及任何境外服务器
- 不涉及任何云端服务

数据是否跨地区传输完全取决于您如何部署和访问您的服务器。我们建议您：
- 在本地局域网内使用以获得最佳安全性
- 如需远程访问，请确保使用 VPN 等安全通道
- 妥善配置服务器防火墙规则

## 11. 本地存储技术

本应用使用设备原生的本地存储技术：
- **SharedPreferences/UserDefaults**：存储应用偏好设置和配置信息
- **加密存储**：敏感信息（如安全密钥）使用加密存储
- **无 Cookie**：本应用不使用网络 Cookie
- **无跟踪器**：不包含任何用户行为跟踪代码

您可以通过以下方式管理本地存储：
- 在应用内清除配置信息
- 通过系统设置清除应用数据
- 卸载应用以完全删除所有本地数据
